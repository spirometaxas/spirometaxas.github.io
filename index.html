MEALSTEAL

MEALSTEAL is a java-built desktop application that informs users of nearby food locations. Facebook API is used to share location information, and Google Maps API is used to clearly display results. I worked with three other students on the project, which was presented at PennApps Spring 2014.

RACER

RACER is a java-built console racing game. The game is composed of three levels, and records user high scores. The game also offers a 2D mode using Java's Graphics2D library, and a 3D mode using rotation, translation, and perspective matricies to achieve the desired affect.
